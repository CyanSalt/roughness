# Avatar

The way of water has no beginning and no end.

## Example

::: details Show Code

```vue
<script lang="ts" setup>
import { RAvatar, RInput, RSpace } from 'roughness'
import { ref } from 'vue'

const name = ref('Cameron')
</script>

<template>
  <RSpace align="center">
    <RAvatar :name="name" />
    <RInput v-model="name" />
    <RAvatar :name="[...name].reverse().join('')" round />
  </RSpace>
</template>
```
