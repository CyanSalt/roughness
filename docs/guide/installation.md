# Installation

### NPM

```shell
npm i roughness
```

### Using from CDN

You can use roughness directly from a CDN via HTML tags:

```html
<link rel="stylesheet" href="https://unpkg.com/roughness/dist/style.css">

<script type="module">
import * as roughness from 'https://unpkg.com/roughness/dist/index.js'
</script>
```
